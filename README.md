🔐 Password Generator

A simple and responsive password generator built using React. It allows users to generate secure passwords based on selected criteria like length, numbers, and special characters.

🚀 Features
Generate random secure passwords
Adjust password length using a slider
Option to include numbers and special characters
Copy password to clipboard with a single click
Clean and responsive UI
🛠️ Tech Stack
React (Hooks)
JavaScript (ES6+)
Tailwind CSS
⚙️ How to Run Locally
git clone https://github.com/SparshRajputtt/Password-Generator.git
cd Password-Generator
npm install
npm run dev
📂 Project Structure
src/
 ├── App.jsx
 ├── main.jsx
 └── index.css
💡 Overview

The application uses React hooks like useState, useEffect, useCallback, and useRef to manage state, generate passwords dynamically, and handle clipboard functionality. Passwords are automatically regenerated whenever the user updates the settings.
